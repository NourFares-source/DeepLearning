📄 AI Resume Architect (Groq + Llama-3)
A high-speed generative AI tool that transforms raw career details into professional, downloadable PDF resumes. This project showcases the intersection of LLM Inference and Automated Document Generation.

🌟 Key Technical Features
Lightning-Fast Inference: Integrated Groq's LPU architecture to achieve sub-second response times using the Llama-3.3-70b-versatile model.

Intelligent Prompt Engineering: Developed a specialized system prompt that acts as an executive resume writer, focusing on action verbs, ATS-friendly formatting, and professional tone.

On-the-Fly PDF Generation: Used the fpdf library to programmatically convert AI-generated text into a structured PDF document available for instant download.

Secure Key Management: Implemented environment-variable-based authentication to ensure API keys are never exposed in the source code.
