## 🎓 AI-Powered MCQ Generator
A tool that leverages Llama 3.3 via the Groq API to instantly transform educational PDFs into structured multiple-choice quizzes.

### 🚀 Key Features
PDF Text Extraction: Uses PyPDF2 to parse complex document layouts.

High-Speed Inference: Integrated with Groq for near-instant question generation.

Modern AI Stack: Built with LangChain’s LCEL (Expression Language) for clean, modular chains.

Export to PDF: Custom-built PDF generation using fpdf for easy sharing.

### 🛠️ Technical Stack
Backend: Python, LangChain, Groq API (Llama 3.3 model).

Frontend: Streamlit.

Logic Architecture: Object-Oriented (OOP)
